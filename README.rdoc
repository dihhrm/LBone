== Labone

<tt>O Labone</tt> é um projeto desenvolvido para fins acadêmicos

O Software foi construído à partir de uma necessidade da cordenação do curso de Fisioterapia da PUC Minas em gerenciar insumos dos seus laboratórios.

O Software tem como principais funcionalidades:

O Cadastro de Alunos com acesso ao laboratório de materiais
O agendamento de horários de uso do laboratório
Cadastro de entrada e saída de materiais do laborátorio
Cadastro de materiais
Dentre outros..
